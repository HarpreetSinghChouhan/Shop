# PHP Shopping Project

A core PHP MVC-based shopping application.

## Features
- User Authentication
- Cart System
- Order Management
- Invoice Generation

## Tech Stack
- PHP 8
- MySQL
- Apache (XAMPP)
- Composer

## Installation
1. Clone repository
2. Run `composer install`
3. Import database
4. Configure database in `config/database.php`
5. Run project via XAMPP
